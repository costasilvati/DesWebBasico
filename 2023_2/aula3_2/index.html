Seletores CSS

Seletores CSS é a forma utlilizada para aplicar estilos a tags ou elementos de uma página HTML. Existem 3 tipos de seletores, são eles:

  1. Seletor de elementos, utilizado para estilizar todas as tags da página que sejam de um certo tipo.
  2. Seletor de id: esse seletor ele estiliza apenas os elementos da página que possuem o id definido; No CSS usa-se # + id.
  3. Seletor de classe: estiliza vários elementos que sejam da mesma classe. Usa-se . nome da classe.

Aplicando CSS

Existem 3 formas possíveis de aplicar CSS a sua página, são elas:

  • Arquivo .css: cria-se um link no head da página para uma folha de estilos (um arquivo de extensão CSS);
  • Aplicando diretamente na tag: utilizando a propriedade style;
  • Tag style: é uma tag onde pode ser escrito todo o estilo css
