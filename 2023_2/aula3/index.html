Funcionamento CSS

Existem 3 formas de colocar estilo CSS na sua página

  1. Associar o estilo ao elemento HTML;
  2. Seção de estilo com a tag style;
  3. Associar um arquivo de extensão .css a página. Utilizando a tag link no head da página.

Seletores CSS

Existem 3 tipos de seletores no CSS, são eles:

  1. Seletores de elemento: onde você apenas usa o nome da tag que quer formatar;
  2. Seletor de ID: escrito com # e o id;
  3. Seletor de classe: utilizado para aplicar estilo em elementos diversos, usa-se: .nome_da_classe;

Cores em CSS

As cores podem ser definidas utilizando o formato RGB (red - green - blue). Neste formato a intensidade de cada cor tem a intensidade definida entre 0 e 255. Onde 0 é 0%, e 255 é 100%.

Exemplo color: rgb(0,255,0) Cor verde em 100%
